# TreeView

File explorer style tree component with expandable/collapsible nodes and keyboard navigation.

```bash
hauk add tree-view
```

## Usage

```tsx
import { TreeView } from "./tui/components/tree-view";

function App() {
  const data = [
    {
      id: "src",
      label: "src",
      children: [
        { id: "index", label: "index.ts" },
        { id: "app", label: "app.tsx" },
        {
          id: "components",
          label: "components",
          children: [
            { id: "button", label: "button.tsx" },
            { id: "input", label: "input.tsx" },
          ],
        },
      ],
    },
    { id: "package", label: "package.json" },
    { id: "readme", label: "README.md" },
  ];

  return <TreeView data={data} onSelect={(node) => console.log(node)} />;
}
```

## Examples

### Basic Tree

```tsx
<TreeView
  data={[
    { id: "1", label: "Documents" },
    { id: "2", label: "Downloads" },
    { id: "3", label: "Pictures" },
  ]}
/>
```

### Nested Tree with Icons

```tsx
<TreeView
  data={[
    {
      id: "root",
      label: "Project",
      icon: "üìÅ",
      children: [
        { id: "src", label: "src", icon: "üìÇ" },
        { id: "pkg", label: "package.json", icon: "üìÑ" },
      ],
    },
  ]}
  defaultExpanded={["root"]}

```

### Without Connecting Lines

```tsx
<TreeView data={data} showLines={false} indentSize={3} />
```

### With Selection

```tsx
const [selected, setSelected] = useState<string>();

<TreeView
  data={data}
  selectedId={selected}
  onSelect={(node) => setSelected(node.id)}
/>;
```

### Disabled Items

```tsx
<TreeView
  data={[
    { id: "1", label: "Available" },
    { id: "2", label: "Locked", disabled: true },
    { id: "3", label: "Another" },
  ]}
/>
```

## Props

| Prop              | Type                       | Default | Description                    |
| ----------------- | -------------------------- | ------- | ------------------------------ |
| `data`            | `TreeNode[]`               | -       | Tree data structure            |
| `selectedId`      | `string`                   | -       | Currently selected node ID     |
| `onSelect`        | `(node: TreeNode) => void` | -       | Callback when node is selected |
| `defaultExpanded` | `string[]`                 | `[]`    | Initially expanded node IDs    |
| `showLines`       | `boolean`                  | `true`  | Show connecting lines          |
| `indentSize`      | `number`                   | `2`     | Indent size in characters      |
| `focusId`         | `string`                   | -       | Focus ID for focus management  |

## TreeNode

```tsx
interface TreeNode {
  id: string; // Unique identifier
  label: string; // Display label
  icon?: string; // Icon to display (emoji or character)
  children?: TreeNode[]; // Child nodes
  disabled?: boolean; // Whether the node is disabled
}
```

## Keyboard Shortcuts

| Key       | Action                |
| --------- | --------------------- |
| `‚Üë` / `k` | Move to previous item |
| `‚Üì` / `j` | Move to next item     |
| `‚Üí` / `l` | Expand node           |
| `‚Üê` / `h` | Collapse node         |
| `Enter`   | Select/toggle node    |
| `Space`   | Select/toggle node    |
| `g`       | Jump to first item    |
| `G`       | Jump to last item     |

## Features

- **Expandable Nodes** - Click or press Enter to expand/collapse
- **Keyboard Navigation** - Full vim-style keyboard support
- **Custom Icons** - Add icons to nodes
- **Connecting Lines** - Visual tree structure with lines
- **Disabled Items** - Mark items as disabled
- **Selection Tracking** - Track and highlight selected item

## Notes

- Arrow keys to navigate, Enter to expand/select
- Supports vim-style j/k/h/l navigation
- g/G for jump to top/bottom
