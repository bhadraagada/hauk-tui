# Stepper

Wizard-style step indicator component for multi-step processes with optional keyboard navigation.

```bash
hauk add stepper
```

## Usage

```tsx
import { Stepper } from "./tui/components/stepper";

function App() {
  const [step, setStep] = useState(1);

  const steps = [
    { title: "Account" },
    { title: "Profile" },
    { title: "Review" },
  ];

  return <Stepper steps={steps} activeStep={step} />;
}
```

### Clickable Steps

```tsx
const [step, setStep] = useState(0);

<Stepper
  steps={steps}
  activeStep={step}
  clickable={true}
  onStepChange={setStep}
/>;
```

## Props

| Prop           | Type                           | Default        | Description                      |
| -------------- | ------------------------------ | -------------- | -------------------------------- |
| `steps`        | `Step[]`                       | -              | Steps to display                 |
| `activeStep`   | `number`                       | -              | Current active step (0-indexed)  |
| `onStepChange` | `(step: number) => void`       | -              | Callback when step changes       |
| `orientation`  | `"horizontal"` \| `"vertical"` | `"horizontal"` | Stepper orientation              |
| `clickable`    | `boolean`                      | `false`        | Allow clicking steps to navigate |
| `focusId`      | `string`                       | -              | Focus ID for focus management    |

## Step

```tsx
interface Step {
  title: string; // Step title
  description?: string; // Step description
  icon?: string; // Optional icon
  optional?: boolean; // Whether step is optional
}
```

## Keyboard Shortcuts

When `clickable` is true:

| Key             | Action              |
| --------------- | ------------------- |
| `‚Üê` / `h`       | Previous step       |
| `‚Üí` / `l`       | Next step           |
| `‚Üë` / `k`       | Previous (vertical) |
| `‚Üì` / `j`       | Next (vertical)     |
| `Enter`/`Space` | Select focused step |

## Step States

| State     | Appearance          |
| --------- | ------------------- |
| Completed | Green checkmark (‚úì) |
| Active    | Highlighted number  |
| Upcoming  | Muted number        |

## Features

- **Step States** - Completed, active, and upcoming states
- **Keyboard Navigation** - Full keyboard support when clickable
- **Dual Orientation** - Horizontal or vertical layout
- **Optional Steps** - Mark steps as optional
- **Custom Icons** - Use custom icons for steps
- **Descriptions** - Add descriptions to steps

## Notes

- Arrow keys to navigate steps, Enter to select
- Supports horizontal and vertical orientations
- Step numbers show 1-indexed (displayed as 1, 2, 3...)
