## Adding Components

Use the CLI to add any component:

```bash
# Add a single component
npx hauktui add button

# Add multiple components
npx hauktui add button dialog toast alert

# Add all components
npx hauktui add --all
```

## Component Structure

Each component consists of:

```
component-name/
├── component-name.tsx   # Main component code
└── index.ts             # Public exports
```

You own the code - feel free to modify, extend, or customize as needed.
