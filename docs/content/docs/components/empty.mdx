# Empty

Empty state placeholder to display when there is no data or content.

```bash
hauk add empty
```

## Usage

```tsx
import { Empty } from "./tui/components/empty";

function App() {
  const items = [];

  if (items.length === 0) {
    return (
      <Empty
        title="No items found"
        description="Try adding some items to get started"
      />
    );
  }

  return <List items={items} />;
}
```

## Props

| Prop          | Type                       | Default     | Description              |
| ------------- | -------------------------- | ----------- | ------------------------ |
| `title`       | `string`                   | `"No data"` | Main title/message       |
| `description` | `string`                   | -           | Description text         |
| `icon`        | `string`                   | `"ğŸ“­"`      | Icon or emoji to display |
| `action`      | `ReactNode`                | -           | Action element (button)  |
| `size`        | `"sm"` \| `"md"` \| `"lg"` | `"md"`      | Size variant             |
