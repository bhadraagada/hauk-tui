# Chart

ASCII chart component for visualizing data in the terminal.

```bash
hauk add chart
```

## Usage

```tsx
import { Chart } from "./tui/components/chart";

function App() {
  const data = [
    { label: "Jan", value: 40 },
    { label: "Feb", value: 65 },
    { label: "Mar", value: 55 },
    { label: "Apr", value: 80 },
    { label: "May", value: 70 },
  ];

  return <Chart data={data} title="Monthly Sales" />;
}
```

### Custom Colors

```tsx
<Chart
  data={[
    { label: "Success", value: 75, color: "green" },
    { label: "Warning", value: 20, color: "yellow" },
    { label: "Error", value: 5, color: "red" },
  ]}
```

### Without Labels

```tsx
<Chart data={data} showLabels={false} showValues={true} />
```

### Without Values

```tsx
<Chart data={data} showLabels={true} showValues={false} />
```

### Custom Width

```tsx
<Chart data={data} maxWidth={50} title="Wide Chart" />
```

## Props

| Prop         | Type                                           | Default            | Description          |
| ------------ | ---------------------------------------------- | ------------------ | -------------------- |
| `data`       | `ChartDataPoint[]`                             | -                  | Chart data           |
| `type`       | `"bar"` \| `"horizontal-bar"` \| `"sparkline"` | `"horizontal-bar"` | Chart type           |
| `title`      | `string`                                       | -                  | Chart title          |
| `maxWidth`   | `number`                                       | `30`               | Max bar width        |
| `showValues` | `boolean`                                      | `true`             | Show values          |
| `showLabels` | `boolean`                                      | `true`             | Show labels          |
| `height`     | `number`                                       | `1`                | Height for sparkline |

## ChartDataPoint

```tsx
interface ChartDataPoint {
  label: string; // Label for data point
  value: number; // Numeric value
  color?: string; // Optional color override
}
```

## Chart Types

### Horizontal Bar

Best for comparing categories with labels.

```
Label1  █████████████░░░░░░░░░ 65
Label2  ████████░░░░░░░░░░░░░░ 40
Label3  ██████████████████████ 100
```

### Vertical Bar

Good for time series with compact labels.

```
    █
  █ █
█ █ █ █
A B C D
```

### Sparkline

Compact trend visualization.

```
▁▃▂▅▄▇▆█▅▃
```

## Sparkline Characters

The sparkline uses 8 levels of block characters:

```
▁ ▂ ▃ ▄ ▅ ▆ ▇ █
```

## Notes

- Supports bar, horizontal-bar, and sparkline types
- Custom colors per data point
- Automatic scaling based on max value
- Labels and values can be hidden
