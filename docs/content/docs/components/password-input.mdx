# Password Input

Secure password input field with toggle visibility option and strength indicator support.

```bash
hauk add password-input
```

## Usage

### Basic Password Input

```tsx
import { PasswordInput } from "./tui/components/password-input";

<PasswordInput
  value={password}
  onChange={setPassword}
  placeholder="Enter password"
/>;
```

### With Toggle Visibility

```tsx
import { PasswordInput } from "./tui/components/password-input";

<PasswordInput
  value={password}
  onChange={setPassword}
  placeholder="Password"
  showToggle
/>;
```

### With Strength Indicator

```tsx
import { PasswordInput } from "./tui/components/password-input";

const getPasswordStrength = (pwd: string) => {
  if (pwd.length < 6) return "weak";
  if (pwd.length < 10) return "medium";
  return "strong";
};

<Box flexDirection="column" gap={1}>
  <PasswordInput
    value={password}
    onChange={setPassword}
    placeholder="Create password"
  />
  <Text
    color={
      getPasswordStrength(password) === "strong"
        ? "green"
        : getPasswordStrength(password) === "medium"
          ? "yellow"
          : "red"
    }
  >
    Strength: {getPasswordStrength(password)}
  </Text>
</Box>;
```

## Props

| Prop          | Type                      | Default | Description                       |
| ------------- | ------------------------- | ------- | --------------------------------- |
| `value`       | `string`                  | -       | Current password value (required) |
| `onChange`    | `(value: string) => void` | -       | Handler when password changes     |
| `placeholder` | `string`                  | -       | Placeholder text                  |
| `showToggle`  | `boolean`                 | `false` | Show visibility toggle button     |
| `disabled`    | `boolean`                 | `false` | Disable input                     |
| `maxLength`   | `number`                  | -       | Maximum password length           |

## Keyboard Shortcuts

| Key         | Action                                    |
| ----------- | ----------------------------------------- |
| `←` `→`     | Move cursor                               |
| `Backspace` | Delete previous character                 |
| `Ctrl+V`    | Toggle visibility (if showToggle is true) |
| `Enter`     | Submit (if onSubmit provided)             |

## Features

- **Masked display** - Characters shown as `•` by default
- **Toggle visibility** - Optional show/hide password button
- **Secure input** - No clipboard history
- **Password strength** - Integrate your own strength checker
- **Keyboard-first** - Standard text input navigation
