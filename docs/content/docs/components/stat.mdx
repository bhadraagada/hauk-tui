# Stat

Statistics display component for showing metrics with optional change indicators.

```bash
hauk add stat
```

## Usage

```tsx
import { Stat, StatGroup } from "./tui/components/stat";

function App() {
  return (
    <Stat
      label="Total Users"
      value="1,234"
      delta={{ value: "+12%", type: "increase" }}
    />
  );
}
```

## Stat Props

| Prop    | Type                                           | Default | Description      |
| ------- | ---------------------------------------------- | ------- | ---------------- |
| `label` | `string`                                       | -       | Stat label       |
| `value` | `string` \| `number`                           | -       | Stat value       |
| `delta` | `{ value: string \| number; type: DeltaType }` | -       | Change indicator |
| `icon`  | `string`                                       | -       | Optional icon    |
| `size`  | `"sm"` \| `"md"` \| `"lg"`                     | `"md"`  | Size variant     |

## StatGroup Props

| Prop           | Type                           | Default        | Description      |
| -------------- | ------------------------------ | -------------- | ---------------- |
| `stats`        | `StatProps[]`                  | -              | Stats to display |
| `direction`    | `"horizontal"` \| `"vertical"` | `"horizontal"` | Layout direction |
| `showDividers` | `boolean`                      | `true`         | Show dividers    |

## Features

- **Delta Indicators** - Show increases/decreases
- **Icons** - Add icons to stats
- **Multiple Sizes** - Small, medium, large variants
- **Stat Groups** - Display multiple stats together
- **Dividers** - Visual separation between stats

## Notes

- Includes Stat and StatGroup components
- Delta shows directional arrows with color coding
- StatGroup supports horizontal and vertical layouts
